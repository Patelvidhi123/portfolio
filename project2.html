Student Placement Portal

Project Description

The Student Placement Portal is a role-based web application designed to simplify the placement approval process within a college. Students can fill out a placement registration form with essential academic and personal details. Once the form is submitted, the data is sent directly to the Principal (Admin Level 1) for verification and approval.

After the Principal approves the submission, the student’s data is automatically forwarded to the specific subject faculty selected by the student (Admin Level 2). Faculty members can then approve, reject, or hold the application based on eligibility criteria. The final approved status is sent to the student via an automated email system.

This project ensures a secure, smooth, and transparent approval workflow while providing clean dashboards for both the Principal and Faculty members to manage placement applications efficiently.

Key Features

  • ✔ Student Placement Registration Form
  • ✔ Role-based Login System (Principal + Subject Faculty)
  • ✔ Principal Level Approval (First Step)
  • ✔ Auto-forwarding of Data to Subject Faculty after Principal Approval
  • ✔ Faculty Level Approval / Rejection / Hold Options
  • ✔ Automatic Email Notification to Students
  • ✔ Admin Panel to View All Submitted Student Records
  • ✔ Status Tracking (Pending / Approved / Rejected)
  • ✔ Secure Data Flow & Validation
  • ✔ Fully Responsive UI with Clean Dashboard Layout

Technologies Used

HTML, CSS, JavaScript, Bootstrap PHP (Backend) MySQL (Database) PHPMailer / SMTP (Email Notification System)
