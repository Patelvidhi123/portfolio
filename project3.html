Toy E-Commerce Website With Admin Panel

Project Description

The Toy E-Commerce Website is a fully functional shopping platform designed for users to browse, purchase, and wishlist toys across multiple age categories. It includes a powerful admin panel where products can be added, updated, deleted, and managed with full control.

The website is divided into two major parts: User Panel and Admin Panel. The User Panel allows customers to view toy collections, check availability, and perform actions like Buy or Add to Wishlist. The Admin Panel helps manage the entire inventory, including images, descriptions, prices, and stock status.

The project features a modern UI, real-time product updates, status toggling (In Stock / Out of Stock), and separate toy categories (14-19 years and 29-39 years), all retrieved from MySQL database tables.

This project now serves as a dummy demonstration of a complete e-commerce flow.

Key Features

  • ✔ Beautiful UI with toy categories for age groups 14-19 and 29-39
  • ✔ Each product displays image, price, description, and stock status
  • ✔ Buy and Wishlist buttons on every product
  • ✔ Product data stored in MySQL database (`inter6` database)
  • ✔ Admin Panel to Add, Edit, Delete products
  • ✔ Toggle Stock Button (In Stock → Out of Stock)
  • ✔ Dynamic product updates without changing frontend code
  • ✔ Secure image upload with file handling
  • ✔ Separate PHP pages for update (`up713.php`, `up2939.php`)
  • ✔ Delete product option (`del713.php`, `del2939.php`)
  • ✔ Toggle status (`togsto713.php`, `togsto2939.php`)
  • ✔ Responsive design with Bootstrap
  • ✔ Clean interface for admin product management
  • ✔ Smooth navigation between user categories

Technologies Used

HTML, CSS, JavaScript, Bootstrap PHP (Backend) MySQL (Database — `inter6`) XAMPP (Local Server)
